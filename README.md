# install Express

 - initialize npm
      ```npm init```
 - install .env
      ```npm i express dotenv```
 - install nodemon
      ```npm i -D nodemon```
 - Run
      ```npm run dev```
 - Install logger
     ```npm install morgan```
 - Install mongoose
     ```npm i mongoose```
 - Slug
     ```npm i slugify```
 - Geo-Coder
     ```npm i node-geocoder```
 - File-Uploader
     ```npm i express-fileupload``` 
 - JWT Token and Encrypt password
    ```npm i jsonwebtoken bcryptjs```
